# Bee.Do Landing Pages

This repository contains two premium, state-of-the-art landing pages for **Bee.Do Insurance Group**, designed for partner recruitment.

## Features
- **Modern Design**: Built using Google Fonts (Plus Jakarta Sans), Tailwind CSS, and hand-crafted CSS for a premium feel.
- **Interactive Simulator**: Dual-slider simulator for calculating potential revenue gains.
- **Glassmorphism**: Elegant use of glass-panel effects, soft shadows, and abstract background elements.
- **Responsive Navigation**: Clean, transparent-to-solid glass navigation bar.
- **Integrations**: Includes Cal.com embed for meeting scheduling and a custom AI Concierge chatbot UI.

## Contents
- `/LP1/beedo-lp_claude.html`: Primary landing page matching the reference design.
- `/LP2/beedo-lp_gemini.html`: Alternative landing page with extended FAQ and chatbot features.
- `/Design/`: Original logo and manual de normas assets.
- `/_old_designs/`: Previous design iterations and reference material.

## Preview
To view the pages locally:
1. Open `LP1/beedo-lp_claude.html` or `LP2/beedo-lp_gemini.html` directly in any browser.
2. Alternatively, serve them using a local web server (e.g., `npx serve`).

## Technology Stack
- **HTML5/JS**: Semantic structure and interactive logic.
- **Tailwind CSS**: Modern styling framework.
- **FontAwesome**: High-quality vector icons.
- **Google Fonts**: Plus Jakarta Sans.
